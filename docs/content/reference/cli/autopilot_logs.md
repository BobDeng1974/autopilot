## autopilot logs

Pipe Logs from the Operator to Stderr/Stdout

### Synopsis




```
autopilot logs <image> [flags]
```

### Options

```
  -f, --follow             Follow logs (Ctrl+C to stop)
  -h, --help               help for logs
  -n, --namespace string   Namespace to which to logs the operator
```
